## 📌 About the Project

Pratham-Chikitse is an offline-first Android application that provides
life-saving first-aid guidance to individuals in emergency situations,
regardless of internet availability. It supports Kannada and English,
making it accessible to rural communities in Karnataka.

---

## 🚨 Problem It Solves

- People panic during emergencies and don't know what to do
- Harmful home remedies (toothpaste on burns, incising snake bites)
  cause more harm
- Rural areas have no internet to search for help
- Medical information is only available in English

---

## ✨ Features

- 🆘 **10 Emergency Modules** — Snake Bite, Burns, Choking,
  Heart Attack, Bleeding, Fracture, Drowning, Poisoning,
  Heat Stroke, Seizure
- 👆 **Step-by-Step Guidance** — Swipeable instructions
  with illustrations
- 🔊 **Audio Mode (TTS)** — Hands-free guidance in Kannada/English
- ✅ **Do's & Don'ts** — Color-coded per emergency
- 📵 **100% Offline** — All content bundled in APK
- 🏥 **Live Hospital Finder** — Google Places API with GPS
- 🧭 **One-tap Navigation** — Opens Google Maps directions
- 📞 **Emergency Numbers** — 108, 100, 101, 1091, 1078
- 🌐 **Bilingual** — Kannada + English instant toggle
- 💾 **Smart Cache** — Shows hospitals even when offline

## 🚀 How to Run

### Prerequisites
- Android Studio (latest version)
- Android device or emulator (API 24+)
- Google Places API key

### Steps

1. Clone the repository
git clone https://github.com/Akanksha-1904-ak/PrathamChikitse.git
2. Open in Android Studio
3. Add your Google Places API key in HospitalFinderScreen.kt
4. Sync Gradle - File then Sync Project with Gradle Files
5. Run on device or emulator API 24 and above
